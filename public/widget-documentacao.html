Widget de Cotações - Alerta de Câmbio

Bem-vindo à documentação do Widget de Cotações do Alerta de Câmbio. Este widget permite que você exiba cotações em tempo real de moedas e criptomoedas em seu site ou blog.

Veja abaixo como o widget aparece em seu site:

Como Incorporar o Widget

Copie e cole o código abaixo em qualquer lugar do seu site onde você deseja que o widget apareça:

<!-- Alerta de Câmbio Widget BEGIN --> <div class="alerta-cambio-widget-container" style="max-width: 100%; margin: 0 auto;"> <div id="alerta-cambio-widget" style="width: 100%;"></div> <div class="alerta-cambio-widget-copyright" style="font-size: 10px; text-align: right; margin-top: 4px; opacity: 0.6;"> <a href="https://alerta.cambiohoje.com.br" rel="noopener nofollow" target="_blank" style="color: #6B7280; text-decoration: none;"> <span>Powered by Cambio Hoje</span> </a> </div> <!-- Primeiro carrega o script --> <script type="text/javascript" src="https://alerta.cambiohoje.com.br/widgets/ticker.js"></script> <!-- Depois inicializa com as configurações --> <script type="text/javascript"> // Inicializa imediatamente, sem esperar pelo DOMContentLoaded AlertaCambioWidget.init({ symbols: "USD/BRL,EUR/BRL,GBP/BRL,CAD/BRL,JPY/BRL,ETH/USD,BTC/USD", theme: "light", bgColor: "#FFFFFF", showLogos: true, showVariation: true, width: "100%", lang: "pt-BR" }); </script> </div> <!-- Alerta de Câmbio Widget END -->

Opções de Personalização

Você pode personalizar o widget alterando os parâmetros na função AlertaCambioWidget.init():

Opção Descrição Valores Possíveis Padrão
symbols Lista de símbolos de moedas separados por vírgula USD/BRL, EUR/BRL, GBP/BRL, CAD/BRL, JPY/BRL, AUD/BRL, CHF/BRL, CNY/BRL, ARS/BRL, BTC/USD, ETH/USD, LTC/USD, XRP/USD, BTC/BRL, ETH/BRL USD/BRL,EUR/BRL
theme Tema do widget light, dark light
bgColor Cor de fundo do widget Qualquer cor hexadecimal ou 'transparent' #FFFFFF
showLogos Exibir logos das moedas true, false true
showVariation Exibir variação percentual true, false true
width Largura do widget Valor em px, % ou qualquer unidade CSS válida 100%
lang Idioma do widget pt-BR, en-US pt-BR

Exemplos de Configuração

Widget com Tema Escuro

AlertaCambioWidget.init({ symbols: "USD/BRL,EUR/BRL,GBP/BRL,CAD/BRL,JPY/BRL", theme: "dark", bgColor: "#333333", showLogos: true, showVariation: true, width: "100%", lang: "pt-BR" });

Widget Apenas com Criptomoedas

AlertaCambioWidget.init({ symbols: "BTC/USD,ETH/USD,LTC/USD,XRP/USD,BTC/BRL,ETH/BRL", theme: "light", bgColor: "#FFFFFF", showLogos: true, showVariation: true, width: "100%", lang: "pt-BR" });

Widget Minimalista (Sem Logos e Variação)

AlertaCambioWidget.init({ symbols: "USD/BRL,EUR/BRL", theme: "light", bgColor: "transparent", showLogos: false, showVariation: false, width: "300px", lang: "pt-BR" });
